## Introduction

The API {Terminology['terminology']['en-US']['PRODUCT']} can be used to access various resources managed by {Terminology['terminology']['en-US']['PRODUCT']}. The API is written following the OpenAPI v3.0.0 specification and provides SDKs in multiple languages, including Java, Go, Python, and Nodejs. </>

## Request type

<>To facilitate batch operations and conditional queries, {Terminology['terminology']['en-US']['PRODUCT']} API supports highly flexible request parameters. </>

To ensure parameter flexibility and usage convenience, and to circumvent `GET` request limitations, we exclusively utilize `POST` requests and transmit data in JSON format.
